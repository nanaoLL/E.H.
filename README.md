# E.H.

니마갤 협전 플렛폼 레츠도로(Let's Doro) 전용 알리미 확장 프로그램입니다.

주요 기능
- 레츠도로 메인 화면에 접속 중일 때, 신규 티켓이 등록되면 Windows 알림이 전송됩니다.
- 전송된 알림을 클릭하면 해당 티켓에 바로 입장 시도 할 수 있습니다.
- 티켓 입장 시 자동으로 참여 코드를 복사합니다.

##

확장 프로그램에 사용된 아이콘 이미지는 원작자의 요청이 있을 시 제거 및 변경됩니다.

그 외 문제 발생 시 해당 확장 프로그램은 별도 공지 없이 수정 혹은 배포 중지될 수 있습니다.
##

# 사용 방법
* Chromium 기반 브라우저에서만 동작합니다.
  * 대상 목록 : 구글 크롬, 네이버 웨일, 마이크로소프트 엣지, etc.
1. https://github.com/nanaoLL/E.H./releases/latest 에 첨부된 **Alarm_버전.zip** 를 다운로드 받습니다.
2. 다운로드 받은 **Alarm_버전.zip**을 임의의 경로에 압축 해제 합니다.
3. 사용하고 있는 인터넷 브라우저에서 확장 관리 페이지에 접근합니다.
  * 각 인터넷 우상단 ··· 메뉴 > 확장 > 확장 관리자 로 접근 혹은 아래 url을 주소창에 입력하여 직접 진입 가능
  * chrome://extensions/
  * whale://extensions/
  * edge://extensions/
4. 해당 페이지에서 개발자 모드를 활성화합니다.
5. 추가된 메뉴 중 "압축해제된 확장 프로그램을 로드"를 클릭합니다.
6. Windows 탐색기가 열리면 사용 방법 2에서 압축 해제한 폴더의 위치에서 **레츠도로 알리미 폴더**를 선택합니다.
7. 설치가 완료되었습니다. 다른 확장 프로그램과 동일하게 사용이 가능합니다.

* 삭제하고 싶을 경우, 다른 확장 프로그램과 동일하게 확장 프로그램 바에서 삭제 또는 사용 방법 3의 관리 페이지에서 삭제할 수 있습니다.

# FAQ

>Q1) 윈도우 알림이 안와요
>
>A1) 키보드의 Window(창문)키 선택 > 설정  > 시스템 > 알림 > "켬"으로 상태 설정되어 있는지 확인
>* 지나간 알림을 확인하고 싶을 경우 작업 표시줄의 날짜 및 시간 영역 클릭 시 확인하지 않은 알림 목록 조회 가능 (Windows 11 기준)
>* <img width="858" height="133" alt="image" src="https://github.com/user-attachments/assets/47ff9db1-b4ca-43a8-84c9-c0483a608c71" />
>
>A2) 레츠도로 사이트의 홈 화면이 브라우저에 켜져있는지 확인
>* Chromium 계열 브라우저는 기본적으로 1시간동안 사용하지 않은 탭을 자동으로 절전 모드로 전환합니다.
>* 만약, 대기 시간이 1시간 이상이 될 것 같음에도 알람을 계속 받아보고 싶다면 아래 경로에서 레츠도로 사이트를 항상 활성화하도록 추가하세요.
>     * 각 인터넷 우상단 ··· 메뉴 > 설정 > 시스템/성능 관련 탭을 선택하거나 하단 url을 주소창에 직접 입력
>     * chrome://settings/system
>     * whale://settings/system
>     * edge://settings/system
>     * 해당 메뉴에서 *성능 > 다음 사이트를 항상 활성 상태로 유지 > 추가* 를 선택한 후 **https://letsdoro.com/** 를 등록합니다.
